# DJ Overlay (Desktop)

Standalone Electron overlay for Rekordbox sessions with:

- draggable/resizable chat and requests panels
- always-on-top transparent window
- request moderation actions (approve/reject)
- metadata-based auto-hide for matched requests

## Run

```bash
cd overlay
npm install
npm run dev
```

## Controls

- `Ctrl+L`: lock/unlock panel layout
- `Ctrl+1`: show/hide chat panel
- `Ctrl+2`: show/hide requests panel
- `Ctrl+I`: toggle click-through mode
- `Ctrl+R`: force refresh requests
- `Esc`: disable click-through
